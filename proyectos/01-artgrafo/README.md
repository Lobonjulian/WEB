# Requerimientos Front-End (Cliente)

## 1. Objetivo
Quiero una web simple y elegante para mostrar mi galería de arte, con imágenes ordenadas y una forma fácil de contactarme.

## 2. Páginas necesarias
- **Inicio:** título, breve texto y un botón de contacto.  
- **Galería:** rejilla de imágenes.  
- **Detalle de obra:** imagen grande + información básica.  
- **Contacto:** formulario para enviarme un mensaje.

## 3. Material que entregaré
- Logo del proyecto.  
- Entre 10 y 20 imágenes en buena calidad.  
- Título principal, textos breves y la descripción de cada obra.  
- Email donde recibir mensajes del formulario.

## 4. Funcionamiento esperado
- Diseño adaptable para móvil y ordenador.  
- Al pulsar una obra debe verse ampliada.  
- El formulario debe enviar mi mensaje y mostrar un aviso de confirmación.  
- Carga rápida y navegación sencilla.

## 5. Elecciones que debo definir
- **Vista de detalle:** ¿modal o página aparte?  
- **Compartir en redes:** ¿activar o no?

## 6. Qué quiero recibir del equipo
- Una versión online para revisar antes de publicar.  
- Una interfaz ordenada, limpia y fácil de usar.
